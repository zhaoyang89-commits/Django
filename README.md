# Game Catalog

Django web application for managing a game catalog.

## Features

- View game list
- View game details
- Search games
- Add new games
- Delete games
- Upload game cover images

## Models

- Game
- Developer
- Platform

Relations:

- One-to-many
- One-to-one
- Many-to-many


## Run

Install dependencies:pip install django
Run migrations:python manage.py migrate
Start server:python manage.py runserver
